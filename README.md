# Calculator

This is a simple calculator project built using HTML, CSS, and JavaScript.  
It can perform basic arithmetic operations like addition, subtraction, multiplication, and division.

## About the Project
This calculator is designed to help users perform basic calculations easily.  
It has a simple user interface and works smoothly on different devices.

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## Features
- Addition, subtraction, multiplication, and division  
- Clear and user-friendly interface  
- Responsive design (works on mobile and desktop)  
- Fast and accurate calculations  

## How to Use
1. Enter numbers using the calculator buttons  
2. Select an operator (+, −, ×, ÷)  
3. Press the equal (=) button to see the result  
4. Use the clear (C) button to reset the calculation  

## Purpose
The purpose of this project is to practice JavaScript logic and improve front-end development skills.
